# Matteo Raciti - Personal Portfolio

A modern, static portfolio website showcasing my work as an HVAC & Refrigeration Engineer and my passion for technology, development, and automation.

## Features

- **Modern Design**: Dark mode, glassmorphism, and neon accents.
- **Responsive**: Fully responsive layout for mobile and desktop.
- **Multilingual**: English and Italian language support.
- **Tech Stack**: HTML5, CSS3, Vanilla JavaScript.

## Hosting

This project is designed to be hosted on GitHub Pages.
